# Pranjal Khare — Portfolio

Personal portfolio website built with **HTML, CSS, and JavaScript**.

## 🌐 Live Website

**[pranjalkhare.me](https://pranjalkhare.me)**

## ✨ Features

- Responsive design (mobile-first, image reordering)
- Dark / light mode with system preference detection
- Resume side panel viewer (lazy-loaded PDF)
- Rotating multilingual greetings (10 languages)
- Letter-by-letter typing animation
- Collapsible skills grid
- Smooth scroll navigation
- Scroll-triggered reveal animations

## 📁 Structure

```
├── index.html          # Single-page layout
├── css/index.css       # Design system + dark mode
├── js/main.js          # Interactions + animations
├── assets/images/      # Profile photo, icons
├── public/resume/      # Resume PDF
├── favicon.svg         # Site icon
├── CNAME               # Custom domain config
└── README.md
```
